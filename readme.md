# Redesign Challenge — Baseline Project

Repository ini merupakan **baseline project** untuk peserta Redesign Challenge sebagai referensi dan contoh struktur pengerjaan project. Kamu tidak diwajibkan mengikuti struktur ini, namun diharapkan dapat memberikan gambaran bagaimana Frontend dan Backend bekerja bersama dalam satu project.

---

## 👥 Struktur Tim

Setiap tim terdiri dari **3 orang** dengan peran masing-masing:

| Peran | Fokus |
|---|---|
| 🎨 UI/UX | Desain tampilan dan pengalaman pengguna |
| 💻 Frontend | Folder `frontend/` |
| ⚙️ Backend | Folder `backend/` |

---

## 🚀 Cara Menggunakan Repository Ini

Repository ini bisa dijadikan **referensi atau contoh** untuk project kamu. Berikut cara clone repository ini:

### 1. Clone Repository
```bash
https://github.com/mauinGit/Redesign-Challenge.git
```

### 2. Masuk ke Folder Project
```bash
cd Redesign-Challenge
```

### 3. Setup Backend & Frontend
Untuk langkah selanjutnya, ikuti panduan di masing-masing folder:
- 👉 [Panduan Backend](./backend/README.md)
- 👉 [Panduan Frontend](./frontend/README.md)

---

## 📌 Catatan

> Repository ini hanyalah **referensi dan contoh**, bukan keharusan. Kamu bebas menggunakan struktur atau teknologi lain yang sesuai dengan kebutuhan timmu.
